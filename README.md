## 💻 Projeto
Esse é um projeto web responsivo de uma capsula do tempo para exibir memeorias de uma linha do tempo.
## 🚀 Tecnologias 
Esse projeto foi desenvolvido durante o NLW da rocketseat com as seguintews tecnologias:

- HTML
- CSS
- GIT e Github
